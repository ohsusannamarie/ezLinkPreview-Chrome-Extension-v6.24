Exclusion list
ezLinkPreview will be disabled for URLs which inclue the listed text
Open in new tab
URLs which inclue the listed text will be opened in a new tab instead of the preview
Auto split list
URLs which inclue the listed text will automatically be loaded split screen
Preview in textonly.in
URLs which inclue the listed text will be previewed in readable mode via textonly.in.
This can work to preview articles from frame-busting news sites (e.g. NYTimes.com).
